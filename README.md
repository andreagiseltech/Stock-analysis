# Análisis de Stock y Sectorización de Depósito
Descripción
Proyecto orientado al análisis, validación y organización de stock, junto con la representación de la distibución física de un depósito.
Se trabajó sobre datos reales para estructurar la información, detectar inconsistencias y generar un modelo claro de interpretación.

Objetivos
- Organizar y depurar datos de stock
- Análizar ditribución por categorías
- Validar consistencia de la información
- Representar la organización física del depósito
- Documentar el proceso completo

  Contenido del repositorio
- **01_Planilla_de_stock.pdf**
  Base de datos procesada
- **02_Resumen_stock.pdf**
  Resumen general con totales y porcentajes
- **03_Informe.pdf**
  Análisis y concluciones
- **04_Plano_deposito.pdf**
  Sectorización del espacio físico
- **05_Detalle_sectores.pdf**
  Descripción de cada sector del depósito

   Herramientas utilizadas
  - LibreOffice Calc
  - LibreOffice Write
  - LibreOffice Draw
  - Git & Github
 
    Enfoque de trabajo
  Se aplicó un enfoque práctico orientado a:
  - orden de la información
  - claridad de la visualización
  - coherencia entre datos y estructura física
